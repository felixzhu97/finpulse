## 各层说明

### 应用层 (`app/`)
- **页面**: Expo Router 认证和主标签页
- **布局**: 导航结构的共享布局组件

### 组件层 (`components/`)
- **自选股**: 股票自选管理组件
- **投资组合**: 投资组合可视化组件
- **账户**: 账户管理和交易组件
- **洞察**: 风险指标和分析组件
- **区块链**: 以太坊钱包集成组件
- **原生**: 高性能原生图表模块
- **UI**: 共享 UI 原语 (MetricCard, SortMenu 等)

### Hooks 层 (`hooks/`)
- **认证**: 认证和令牌管理 hooks
- **投资组合**: 投资组合数据获取 hooks
- **市场**: 市场数据和行情流 hooks
- **风险**: 风险指标和 VaR 计算 hooks
- **账户**: 账户数据和偏好设置 hooks
- **区块链**: Web3 和钱包 hooks

### Store 层 (`store/`)
- **authSlice**: 用户认证状态
- **quotesSlice**: 实时股票行情
- **portfolioSlice**: 投资组合数据缓存
- **preferencesSlice**: 用户偏好设置
- **web3Slice**: 以太坊钱包状态

### Lib 层 (`lib/`)
- **api/**: 所有后端服务的 REST API 客户端
- **services/**: QuoteStreamService, Web3Service
- **network/**: HTTP 客户端, 认证桥接, 配置
