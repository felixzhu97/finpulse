# 商业模式与盈利模式

本文档描述组合分析平台的收入来源与商业化策略。

## 收入来源

### 1. 订阅 / SaaS

| 档位 | 目标用户 | 功能 |
|------|----------|------|
| Free | 普通用户 | 基础组合视图、有限历史 |
| Pro | 活跃投资者 | 完整分析、AI 风险/VaR、情感分析、历史数据 |
| Premium | 高净值用户 | 优先支持、高级 AI、定制报告 |

### 2. 交易佣金

- 用户通过平台对接券商交易时，与券商分成。
- 按笔收费或按成交金额比例收费。

### 3. 资产管理费

- 对托管组合或智能投顾收取 AUM 管理费。
- 示例：年化 0.25%–0.50% 资产管理规模。

### 4. 增值服务

- AI 分析：VaR、欺诈检测、情感分析、预测。
- 高级风险指标与组合优化。
- 付费行情、估值数据等。

### 5. 数据与 API

- 向第三方出售行情、风险指标或分析数据。
- 开发者 API 按调用量或订阅收费。

### 6. 白标 / B2B

- 向银行、券商、财富管理机构授权平台能力。
- 按年或按用户数的授权费。

### 7. 广告与导流

- 券商、基金、理财产品推广位。
- 合规前提下，通过导流获得返佣。

## 目标客群

| 客群 | 需求 | 适合的盈利方式 |
|------|------|----------------|
| 散户投资者 | 组合跟踪、洞察 | 订阅、广告 |
| 活跃交易者 | 实时行情、风险工具 | 订阅、佣金 |
| 机构 | API、白标 | B2B、数据/API |
| 投顾 | 客户组合、报告 | 白标、高级功能 |

## 定价考量

- 提供免费档以提升获客。
- 清晰的 Free → Pro → Premium 价值阶梯。
- B2B、API 提供用量或批量折扣。
- 按地区做本地化定价与合规。
